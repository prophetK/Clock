# Clock
HTML/JavaScript/CSS

# 说明
尝试网页上的动态效果，选择canvas2D，模拟时钟运行  
设定计时器setInterval，每过一秒(1000ms)获取本地时间  
经时/分/秒公式计算得到三者的角度，以及该时间所有指针对应的位置
